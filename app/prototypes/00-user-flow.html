Flow 1 — New Visitor (anonymous)

1.1
Landing Page — Idle
Full-screen treasure box (random public box).
Pulsing "pull the drawer" hint.
Faint "treasure box" in corner.
Nav hidden.
visible: box + idle hint only
nav: hidden (hover top-right to reveal)
nav items: gallery · sign in
▼
1.2
What does the visitor do?
A) Pull the drawer / interact with items
B) Scroll down without interacting
C) Hover top-right corner
D) Do nothing (idle)
path A: → 1.3 (interacted state)
path B: → 1.5 (below fold, no scroll hint shown)
path C: nav fades in temporarily
path D: hint keeps pulsing
▼ path A: interacts
1.3
Landing Page — Interacted
Items spill out / physics active.
"Pull the drawer" hint fades out.
Interaction hints appear: "drag" "hold" "tilt".
Nav fades in: gallery · sign in.
"Scroll to explore" appears at bottom.
trigger: click/drag on drawer or items
new elements: nav, interaction hints, scroll indicator
corner title: brightens
▼
1.4
Explore the box
Drag items around (physics).
Long-press item → story card overlay.
Tilt phone → items shift with gravity.
Click item link → opens URL.
story card: shows item label, story text, optional link
dismiss: click outside or close button
▼ scrolls down
1.5
Below the Fold — About
ASCII-framed tagline: "a tangible memory box for the web".
Brief description paragraph.
always visible: accessible by scrolling regardless of interaction state
▼
1.6
Make Your Own Box — CTA
ASCII-framed "make your own treasure box".
Short copy + "make your own box →" button.
button action: navigates to /editor
if not signed in: editor shows sign-in prompt first
▼
1.7
Public Gallery
"public boxes" header with count.
Grid of gallery cards (mini box previews).
Each card: numbered, title, item count, background color.
click card: opens /embed?box={id} in new tab
empty state: "no public boxes yet — be the first"
▼
1.8
Footer
"physics · memories · webring"

Flow 2 — Sign In (anonymous → authenticated)

2.1
Click "sign in" (nav) or "make your own box" (CTA)
Both paths lead to authentication.
nav "sign in": direct auth popup
"make your own box": goes to /editor which gates on auth
▼
2.2
Google OAuth Popup
Firebase Google sign-in popup.
One click, no forms.
success: → 2.3
cancel/fail: stays on current page
▼
2.3
Where did user sign in from?
A) From nav "sign in" on landing page
B) From "make your own box" button
C) From /editor page directly
path A: nav updates: "sign in" → "my box" · "sign out"
path B & C: redirects to editor
▼
2.4
Landing Page — Signed In
Same as before but nav shows:
gallery · my box · sign out
"my box": links to /editor
sign out: returns to anonymous state

Flow 3 — Creator (authenticated, in editor)

3.1
Editor — Items Tab
Upload images → auto background removal.
Edit label, link, story per item.
Reorder / delete items.
first visit: empty state "no items yet — upload your first treasure"
▼
3.2
Editor — Config Tab
Drawer label, background color, sound preset, volume.
Public toggle: [ private ] [ public ]
"your box will appear in the public gallery"
new: isPublic toggle (opt-in)
default: private
▼
3.3
Editor — Embed Tab
Copy iframe code.
Copy inline script code.
Share link.
▼
3.4
Editor — Preview Tab
Live 700x700 preview of your treasure box.
Real-time updates as you edit.

Flow 4 — Viewing a Public Box

4.1
Entry point
A) Click gallery card on landing page
B) Direct link: /embed?box={userId}
C) Embedded iframe on external site
gallery click: opens in new tab
▼
4.2
Embed Page (/embed)
Full treasure box experience.
Read-only, no auth required.
Physics, sounds, stories all work.
No editor access, no nav.
error states: "box not found", "missing box id"

Landing Page — State Summary

Idle + Anonymous
Full-screen box, items inside drawer.
"Pull the drawer" pulsing.
No scroll indicator.
nav (on hover): gallery · sign in
Interacted + Anonymous
Items spilled, physics active.
Interaction hints: drag, hold, tilt.
"Scroll to explore" visible.
nav (always): gallery · sign in
Interacted + Signed In
Same as interacted + anonymous
but nav updates.
nav (always): gallery · my box · sign out

Navigation States

Anonymous
gallery · sign in
"sign in": triggers Google OAuth popup
Signed In
gallery · my box · sign out
"my box": goes to /editor
"sign out": returns to anonymous
Visibility
Hidden on load.
Revealed on: hover top-right OR after interaction.
Stays visible once revealed.
position: fixed top-right corner
